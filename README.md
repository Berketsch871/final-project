# Regression & Sigmoid Playground

Open this folder in VS Code and run Live Server (or open `index.html` in a browser) to use the interactive playground.

Controls:
- `Weight` and `Bias`: move the regression line (or the sigmoid's linear input).
- `Learning Rate`: step size for gradient descent.
- `Sigmoid`: toggle classification mode (shows sigmoid curve and computes binary cross-entropy + accuracy).
- `Randomize Data`, `Auto Gradient Descent`, `Stop`, `Reset Model`.
